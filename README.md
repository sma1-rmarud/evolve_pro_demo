# ES activity prediction using Deep Learning

## Installation

```bash
conda create -n bio python=3.10 -y
conda activate bio

git clone --recurse-submodules https://github.com/sma1-rmarud/protein_demo.git
cd protein_demo

pip install -e .
```

## Usage

To run the GUI server, execute: 
```bash
./run_gui.sh
```
